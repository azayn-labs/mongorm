# MongORM

Started as a pet project, MongORM is a simple Object-Relational Mapping (ORM) library for MongoDB. It provides a convenient way to interact with MongoDB databases using Go structs and methods.

## Features

- Simple and intuitive API for defining models and performing CRUD operations.
- Support for basic querying and filtering.
- Easy integration with existing Go applications.
- Lightweight and minimalistic design.

## Installation

To install MongORM, you can use the following command:

```bash
go get github.com/CdTgr/mongorm
```
